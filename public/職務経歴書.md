### 実績・取り組み
- PHP(航空)、Vue(レンタカー)、React(ホテル)など、それぞれ異なる技術スタックで構成されたサービスたちのユーザー体験を統一するため、WebComponentsを使用してサイト全体のグローバルナビゲーションシステムを設計・開発

- Google Tag ManagerとdataLayer構築によるWeb分析システム実装
- CSRページをSSR,SSG(ISR)に変更して初期表示速度の改善とSEOパフォーマンスの向上
- Core Web Vitals改善 - FCP, LCP, CLS, FID改善、イメージ最適化でUX向上、Lighthouseスコアを「改善が必要（黄色）」から「良好（緑色）」へと引き上げ
- リンク、メタデータ設定とcanonical tagを利用したURL正規化実装
- 海外ホテル検索メインページ作成とコンテンツ追加
- ディレクター、外部SEOコンサルタント、デザイナーと協業してSEO専用のテーマページ作成
- 外部レビューシステム(yotpo)連携のレビューページ作成
- 外部API連携によるホテルアメニティデータ収集とSEO最適化FAQ作成を自動化
- Google BigQueryからデータを抽出し、フロントエンドで加工・表示するシステム構築
- バックエンドチームと連携してSEO用のAPI設計、改修

### 言語・FW等
React, Next.js, TypeScript, Styled Components, Web Components, Redux, Lit Elements, Cypress, Storybook, GCP, BigQuery, Google Optimize, Google Tag Manager, Docker, yotpo, GitHub, Figma
